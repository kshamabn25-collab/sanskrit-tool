Installation

Windows applies several security checks to downloaded apps. Follow these steps in order to get through them.

1

Download — allow the file in your browser

When downloading the .exe, your browser may flag it as an uncommon file:

Chrome

  1. Click the download in the bottom bar — Chrome may show "This file isn't commonly downloaded".
  2. Click the arrow (›) or the three-dot menu next to the file name.
  3. Select Keep.

Microsoft Edge

  1. Edge may show a download warning in the sidebar or bottom bar.
  2. Click See more (or the three-dot menu on the download).
  3. Click Keep anyway.
2

Run the installer — bypass SmartScreen

Windows Defender SmartScreen may show a blue warning screen when you run the installer. This happens because the app doesn't yet have a paid Extended Validation certificate.

  1. Double-click the downloaded Sanskrit-Tool-windows.exe.
  2. When the blue SmartScreen screen appears, click More info (the small text link at the bottom-left).
  3. A Run anyway button appears — click it.
  4. If a User Account Control (UAC) prompt appears asking for permission, click Yes.
  5. The installer runs and Script Tool is installed.
3

Launch and find the tray icon

  1. After installation, Script Tool starts automatically.
  2. Look for the ॐ icon in the system tray (bottom-right of your screen).
  3. If you don't see it, click the ^ arrow in the taskbar to expand hidden tray icons.
  4. Right-click the icon to access all features, or use the keyboard shortcuts below.

How to use Script Tool

A

Inline IME — type in any app

Keystrokes intercepted system-wide. A small popup appears near your cursor showing the live conversion.

1

Toggle IME on — Alt+Shift+M

A small candidate popup appears near your mouse cursor showing your active scheme (e.g. ITRANS ▶ Active). Click into any text field in any app.

2

Type phonetic text

Start typing — e.g. namaskaara. The popup near your cursor shows namaskaara → नमस्कार live as you type. Nothing appears in the app yet.

3

Commit the word — Space or Enter

Press Space — नमस्कार is pasted directly into the text field via clipboard. Press Enter to insert with a newline. Continue typing the next word immediately.

4

Switch script or scheme

Press Alt+Shift+O to cycle through all 7 scripts. Press Alt+Shift+1/2/3 to switch between ITRANS, Harvard-Kyoto, and SLP1.

5

Exit IME — Esc or Alt+Shift+M

Press Esc to cancel the current word and exit. Or press Alt+Shift+M again to toggle off.

B

Offline Converter

Side-by-side Indic script + IAST output. Copy with one click.

1

Open the converter — Alt+Shift+I

Press the shortcut from any app, or right-click the ॐ system tray icon and choose Open Script Tool.

2

Choose scheme and script

Select your input scheme (ITRANS / Harvard-Kyoto / SLP1) and output script using the buttons in the window.

3

Type and copy

Type phonetic text in the input box. The Indic script and IAST outputs appear instantly. Click Copy on whichever you need and paste it anywhere.

C

Quick Panel

Floating overlay — type once, paste directly into the active app.

1

Open the panel — Alt+Shift+S

A small floating panel appears over whatever you're working in.

2

Type and preview

Type your word phonetically and see the Devanagari preview instantly.

3

Paste — Enter

Press Enter or click Paste ↵. The converted text is pasted into the app that was active before the panel opened. Press Esc to close without pasting.
