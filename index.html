How it works

A

Inline IME — type anywhere macOS

1

Install & grant Accessibility

Download and run the installer. On first launch go to System Settings → Privacy & Security → Accessibility and enable Script Tool — this lets it intercept keystrokes system-wide.

2

Toggle IME with ⌥⇧M

A slim status bar appears at the top of your screen showing the active scheme and script. Click into any text field in any app.

3

Type, see live preview, commit

Type namaskaara → the bar shows namaskaara → नमस्कार live. Press Space — the converted word is inserted directly. Continue typing the next word immediately.

4

Switch script or scheme anytime

Press ⌥⇧O to cycle through all 7 scripts, or ⌥⇧1/2/3 to switch between ITRANS, Harvard-Kyoto, and SLP1 — without leaving your app.

B

Offline Converter — copy & paste macOS + Windows

1

Open the converter (⌥⇧I)

Press the hotkey anywhere to bring up the full converter window, or click the tray icon.

2

Type phonetic text

Type using ITRANS, Harvard-Kyoto, or SLP1. The output in your chosen Indic script and IAST appear side by side instantly.

3

Copy and paste anywhere

Click Copy on either output — Indic script or IAST — and paste into any document, email, or message.
